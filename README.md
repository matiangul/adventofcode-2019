# Advent of Code 2019

## Environment

```
python 3.7.4
pytest 5.3.1
```

## How to run program for selected day

Command looks same for each day, as an example here it is day 1:

```
python day_1.py < input/day_1.txt
```

## How to run tests

Command looks same for each day, as an example here it is day 1:

```
pytest -s test_day_1.py < input/day_1.txt
```
